# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-07

### Added
- 🎉 初始版本发布
- ✅ Web 版本 - 支持浏览器直接访问
- ✅ Windows 便携版 - 无需安装即可运行
- ✅ Linux AppImage - 通用 Linux 格式
- ✅ Linux DEB - Debian/Ubuntu 安装包
- ✅ Linux TAR.GZ - 通用压缩包
- ✅ Android APK - 支持 Android 和鸿蒙系统
- 🔗 去中心化 P2P 通信 - 无需服务器
- 💬 实时文本聊天
- 📁 文件传输功能
- 👥 局域网用户自动发现
- 📱 响应式设计 - 完美适配移动端

### Technical
- 使用 BroadcastChannel + WebRTC 实现 P2P 通信
- Electron 桌面端支持
- Capacitor 移动端支持
- GitHub Actions 自动化构建
- 阿里云镜像源加速 Android 构建
