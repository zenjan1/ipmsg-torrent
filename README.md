# IPMsg Torrent - 去中心化聊天软件

一个基于 Torrent 技术的去中心化聊天软件，类似 IP Messenger 和 飞秋 的功能，但不需要中心服务器。

## 功能特性

- 🔗 去中心化 P2P 通信，无需服务器
- 💬 实时文本消息聊天
- 📁 基于 WebTorrent 的文件传输
- 👥 局域网用户自动发现（UDP 广播）
- 🎨 现代化的用户界面
- 🖥️ 跨平台支持（Windows、macOS、Linux）

## 技术栈

- Electron - 桌面应用框架
- WebTorrent - P2P 文件传输
- Node.js - 后端运行时
- UDP Socket - 局域网发现和消息传输

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

### 生产构建

```bash
npm run build
```

## 使用方法

1. 启动应用后，在顶部设置你的用户名
2. 同一局域网内的其他用户会自动出现在左侧用户列表中
3. 点击用户开始聊天
4. 输入消息并回车或点击发送按钮
5. 点击📎按钮选择文件进行传输

## 工作原理

### 用户发现

- 应用启动后通过 UDP 广播在局域网（2425端口）宣布自己的存在
- 每隔3秒广播一次心跳包
- 收到广播的用户会自动添加对方到在线列表

### 消息传输

- 使用 UDP 协议（2426端口）进行实时消息传输
- 支持文本和文件两种消息类型

### 文件传输

- 使用 WebTorrent 协议进行文件分享
- 发送方创建种子文件并生成 magnet 链接
- 通过聊天消息发送 magnet 链接给对方
- 接收方通过 magnet 链接下载文件

## 注意事项

- 目前仅支持局域网内通信
- 需要确保防火墙允许 2425 和 2426 端口的 UDP 通信
- 文件使用 WebTorrent 协议传输，可能需要一定时间建立连接
